# Mechanics  Pet Dog Desk Robot


This project is an open-source hardware initiative to create a low-cost, quadruped pet robot using the ESP32-C6 microcontroller. The robot features 4 SG90 servo motors for movement, an OLED display for visual feedback, and connectivity options including Wi-Fi 6 and Bluetooth 5. The design is modular and 3D-printable, allowing for easy customization and repair. The project aims to provide an accessible platform for robotics enthusiasts to learn and experiment with quadruped locomotion and IoT integration.
